## 1. DOMAIN: brooklynpsychiatricsociety.org (~$7/year)

**Recommended registrar: Porkbun** (cheap, clean UI, free WHOIS privacy)

1. Go to https://porkbun.com
2. Search `brooklynpsychiatricsociety.org`
3. Add to cart → Checkout
4. Create a Porkbun account (email + password)
5. Pay (~$6.88 first year)
6. After purchase, go to **Domain Management** → click your domain
7. Go to **DNS Records**
8. Add these records:

| Type | Host | Value |
|------|------|-------|
| CNAME | `www` | `stephancarlson.github.io` |
| A | (blank/`@`) | `185.199.108.153` |
| A | (blank/`@`) | `185.199.109.153` |
| A | (blank/`@`) | `185.199.110.153` |
| A | (blank/`@`) | `185.199.111.153` |

9. Go to https://github.com/stephancarlson/brooklyn-psychiatric-society/settings/pages
10. Under **Custom domain**, enter: `www.brooklynpsychiatricsociety.org`
11. Click **Save**
12. Wait 5-30 min for DNS propagation
13. Check **Enforce HTTPS** when it appears
14. Test: visit `www.brooklynpsychiatricsociety.org`

## 2. EMAIL FORWARDING (free with Porkbun)

1. In Porkbun → Domain Management → **Email Forwarding**
2. Add: `info@brooklynpsychiatricsociety.org` → forward to `lindabps@aol.com`
3. Optional: `membership@` → same, or a different inbox
4. Test by sending an email to `info@brooklynpsychiatricsociety.org`

## 3. GOOGLE VOICE (free phone number)

1. Go to https://voice.google.com
2. Sign in with a Google account (personal is fine)
3. Click **Search** for a number
4. Search area code `718` or `347` (Brooklyn)
5. Pick a number → **Verify** with your real cell
6. Your new BPS number is ready — add it to the website and all platform bios
7. Set up voicemail greeting: "You've reached the Brooklyn Psychiatric Society..."
